YouTube RAG Chatbot
This project implements a Retrieval Augmented Generation (RAG) chatbot that can answer questions based on the transcript of a YouTube video. It uses LangChain for orchestrating the RAG pipeline, the YouTube Transcript API to fetch video transcripts, and Groq's LLMs for generating responses.

Features
Fetches YouTube video transcripts.
Chunks transcripts and creates embeddings for efficient retrieval.
Builds a FAISS vector store for fast similarity search.
Uses a ChatGroq LLM to generate answers based on retrieved context.
Provides an interactive ipywidgets based UI to easily query any YouTube video.
Setup and Usage
1. Install Dependencies
Ensure you have all the necessary libraries installed by running the first cell in the notebook:

!pip install -q \
langchain \
langchain-community \
langchain-groq \
youtube-transcript-api \
faiss-cpu \
sentence-transformers \
langchain-text-splitters
2. Configure API Key
This project uses the Groq API for the Large Language Model. You need to set your GROQ_API_KEY in Colab secrets.

Go to the "🔑" icon in the left panel of Colab, click on "Secrets", and add a new secret named GROQ_API_KEY with your Groq API key as its value.

The notebook will automatically load this key and set it as an environment variable.

3. Run the Notebook Cells
Execute the cells sequentially to:

Load the API key.
Define a YouTube video_id.
Fetch and process the transcript.
Create embeddings and build the FAISS vector store.
Set up the RAG pipeline with a PromptTemplate and ChatGroq LLM.
Run a sample query.
4. Use the Interactive UI
An interactive UI is provided using ipywidgets for easy Q&A. Scroll to the last code cell and run it. You will see input fields for:

Video ID: Enter the ID of the YouTube video you want to query (e.g., FwOTs4UxQS4).
Question: Type your question related to the video content.
Click the "Get Answer" button to receive a response generated by the RAG chatbot based on the video's transcript.

Project Structure
Environment Setup: !pip install commands.
API Key Configuration: Loading GROQ_API_KEY from Colab secrets.
Data Ingestion: Using YouTubeTranscriptApi to fetch transcripts, RecursiveCharacterTextSplitter for chunking, HuggingFaceEmbeddings for embeddings, and FAISS for vector storage.
RAG Pipeline: ChatGroq for LLM, PromptTemplate for structured prompts, and a retriever for context retrieval.
Interactive UI: ipywidgets for user interaction.
